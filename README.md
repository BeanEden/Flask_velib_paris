# Vélib’ Dashboard

## Description
Dashboard pour visualiser les stations Vélib’ de Paris avec Flask et MongoDB.  
Permet de filtrer les stations, voir les disponibilités en vélos et bornes, et afficher un graphique d’évolution.

## Prérequis
- Docker & Docker Compose
